# Student Portal Demo 🎓

## 📌 Overview
Student Portal Demo is a web-based application designed to simplify student academic management. It allows students to access course details, view grades, submit assignments, and receive announcements from faculty.

## 🚀 Features

### Student
- User Registration & Login
- View Profile
- Enroll in Courses
- View Marks & Grades
- Submit Assignments
- View Announcements

### Admin/Faculty
- Manage Courses
- Upload Marks
- Post Announcements
- Manage Students

## 🛠️ Tech Stack
- Backend: Django (Python)
- Frontend: HTML, CSS, Bootstrap
- Database: SQLite

## 🎯 Objective
The goal of this project is to build a simple and functional student management system while learning full-stack web development using Django.

## 📈 Future Enhancements
- Email notifications
- Chat system between students and faculty
- Dashboard analytics
- File upload support
